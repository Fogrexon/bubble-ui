# コーディング規約

- TypeScript を使用
- スタンドアロン関数はアロー関数、クラスは標準メソッド構文
- 公開関数/クラスは JSDoc でコメント
- コメントは「現在のコードの状態」のみ記述（過去との比較禁止）
- **設計方針**:
  - **インターフェースの活用**: 明確にインターフェース（`interface`）を切り、モジュール間の依存関係を整理して、テスト容易性（テスタビリティ）の高いコードを書く
  - **ライブラリとしての配布**: このリポジトリは最終的に外部向けの「ライブラリ」として配布される予定。それを踏まえ、公開インターフェースやモジュールのカプセル化を意識した設計を行う
- 依存関係:
  - 型チェック: TypeScript
  - テスト: Jest
  - 描画: Pixi.js v8
