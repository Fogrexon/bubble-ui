    • reconcile(newVNode: null | VNode, oldVNode: null | VNode): void

      Performs the reconciliation (diffing and committing) of the virtual DOM tree. If the root newVNode is a component, it resolves the component first. Then, it calculates the differences (WorkUnits) and applies them via the Committer.

      Parameters
